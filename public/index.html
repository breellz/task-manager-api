Task Manager API

This is a task manager api built with NodeJs. Users can sign up, login, edit their details, create tasks, update them, view and mark as completed. Users get email notifications on account creation and deletion. It also supports profile pictures, pagination to manage fetching tasks
Documentation Github

  • User features

  • Sign up, Login, LogOut
  • Edit profile details, upload profile picture, delete account and tasks
  • create, edit, fetch and delete tasks
  • Fetch tasks with specific attributes like completed, sorting and pagination
  • Get emails on signup and account deletion
